1. pwd

pwd ＃找現在位置

找現在位置

2. cd

cd     #進入其他資料夾

cd ..  #回到上一層資料集

cd ~   #回到目錄

cd /   #回到根目錄（電腦最底層）

change directory

記得cd後要空格

如果要一次進入多個資料夾，可以加/

例如 cd desktop/example

也可以打cd之後直接將檔案拉進終端機視窗

3. ls

ls      #列出目錄

ls -a   ＃列出隱藏資料

ls -l   ＃列出詳細資料

ls -la  ＃列出隱藏目錄以及詳細資料

ls *.html  ＃列出所有html檔案

＊是全部的意思

4. mkdir

mkdir new_folder_name #建立新目錄

也就是建立新資料夾

在mkdir後加空格再加資料夾名稱

5. touch

touch  new.docx #建立檔案

如果有相同的檔案，只會輕輕碰一下，內容不變，但修改的時間會改

可一次加多個檔案

例如touch new.docx new2.docx

6. cp

cp copy.docx newcopy.docx  #複製檔案
cp -r copyfolder newfolder #複製資料夾

先放要複製的檔案，在打新複製的檔案名

7. mv

mv new.docx ./folder2 #移動new.docx至folder2
mv new.docx n2.docx   #更改new.docx檔名為n2.docx

如果要移到下一層的資料夾，可以用上面的方法

記得加點./folder2

如果要移到上一層的資料夾，可以用users/usersname/...

8. rm

rmdir  #刪除空資料夾
rm -rf #刪除檔案或資料夾

rmdir如果資料夾內有檔案就不會刪除

如果要刪除的檔案有特殊字元會空格可以用單引號'括起來

例如rm -rf 'a new document.docx'

9. clear

clear  #清除terminal上面內容

用於清除terminal上的內容，但往上滑還是可以看到舊內容

另外，history可以看指令歷史紀錄

10. whoami

whoami  #檢視電腦名稱

用來確認現在使用的使用者名稱

只是我現在還不知道用在哪裡XD

11. echo

echo hi #印出字串hi
echo hi >> new.txt  #將hi加入new.txt中

如果沒有該檔案，就會新增一個檔案

12. cat

cat new.txt #terminal顯示new.txt

concatenate(串接)

13. less

less new.txt #以分頁形勢顯示在terminal

14. grep

grep hi new.txt  #在new.txt抓關鍵字hi

15. date

date  #時間

16. wc

wc new.txt #計算new.txt當中的行數、單字數、字元數
wc -l new.txt  #line
wc -w new.txt  #word
wc -c new.txt  #character

也可以一次計算多個檔案
